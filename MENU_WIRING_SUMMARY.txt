OVERVIEW
--------
The GCodeKit4 menu system has been fully implemented with functional menu items
and proper event handlers wired up to application functionality. The implementation
follows patterns from Universal G-Code Sender (Java reference).

FILES MODIFIED
--------------
1. src/ui.slint
   - Replaced generic menu callbacks with specific action callbacks
   - Implemented dropdown menus with File, Edit, View, and Help items
   - Added hover effects for menu items (background color changes)
   - Menu dropdowns use conditional rendering with Slint's if statements
   - Menus close automatically after item selection

2. src/main.rs
   - Implemented 5 menu callbacks:
     * menu-file-open(): Opens G-Code file dialog
     * menu-file-exit(): Gracefully closes application, disconnects machine
     * menu-edit-preferences(): Opens preferences/settings dialog
     * menu-view-fullscreen(): Toggles fullscreen mode
     * menu-help-about(): Displays about dialog with version info

MENU STRUCTURE
--------------
File Menu:
  - Open File      → Opens file dialog to load G-Code
  - Exit           → Cleanly closes application

Edit Menu:
  - Preferences    → Opens settings dialog

View Menu:
  - Fullscreen     → Toggles fullscreen mode

Help Menu:
  - About          → Shows application information
